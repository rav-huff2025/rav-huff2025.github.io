Главная Коллекция марок Huffmory Game

Правила лотереи Pufftery Game:

  1. Перед вами 6 закрытых марок - откройте их все
  2. Если найдёте 3 одинаковые марки - вы выиграли!
  3. Особые призы за:
    • собранный сет из 3-х марок с выпускниками факультета Пуффендуй
    • собранный сет из 3-х марок с "Пуффендуйскими плюшками"
    • собранный сет из 3-х марок из категории "Символы факультета"
  4. Нажмите "Начать заново" для новой попытки
